Version 2.0 (06.10.2010)
-----------------------------------------------------
* Поправлены баги:
  - Исключение при удалении нод, сломавшееся в процессе доделки беты
  - Критическая ошибка сохранения, приводившая файл данных в негодность
  - В новом редакторе нельзя было сменить шрифт и его размер при пустом документе
  - При переименовании ноды в статус-строке оставалось старое имя
  - Устранена существенная утечка памяти (FlowDocument)
* Убран слишком большой отступ между абзацами в новом редакторе
* Добавлена опция загрузки последнего файла при запуске и история недавних документов

Version 2.0 beta (30.09.2010)
-----------------------------------------------------
* Реализована поддержка плагинов
* Новый формат файла, автоматическая конверсия из старого формата
* Новый текстовый редактор (WPF)
* Форматирование сразу всех выбранных нод
* Добавлены выравнивание абзацев, отступы, списки, нижние и верхние индексы
* Поправлены баги:
  - Исключение при попытке нажимать стрелки при пустом дереве
  - Исключение при удалении нод
  - Некоторые ошибки при редактировании в старом редакторе
  - Размер файла при пересохранении никогда не уменьшался
* При получении фокуса деревом текущая нода больше не переходит в режим редактирования
* Добавлен экран настроек
* Автоматическая запись разрешений в реестр при первом запуске
* Прочие исправления и добавления

Version 1.2 (03.03.2010)
-----------------------------------------------------
* Поправлены баги:
  - Основные элементы и их скроллбары прятались под статус-бар
  - Перемещение ноды вверх и вниз через Ctrl+стрелка
  - При перетаскивании нод с жирным текстом некорректно перерисовывались эти ноды
  - Кнопка удаления перехватывалась из всех элементов
  - Выбранная нода оставалась выбранной даже после удаления
  - Удаление не проверяло наличия выбранных нод
  - При нажатии отмены в диалоге "сохранить перед выходом" приложение закрывалось
  - Некорректно сохранялись цвета нод
  - Не всегда правильно определялся факт изменения файла
* Иконка сохранения неактивна, если файл не изменялся
* При переходе по истории раскрывается нода, содержащая целевую
* Указывается количество удаляемых узлов
* Назначение цвета стало удобнее
* Сделаны горячие клавиши для стилей текста

Version 1.2 beta

* Поправлены разные мелкие баги

* Перетаскивание на форму или в дерево файла откроет его
* Пофиксено сохранение размеров окна в реестр для минимизированного окна
* Пофиксено местами закрытие модифицированного документа без вопросов
